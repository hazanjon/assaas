SASSAS!

Better known as 'Syntactically Awesome Style Sheets Automated Service' is a tool designed to help developers with the trails of converting SASS & SCSS.

Get Your API Key

Login using Paypal OpenID

Direct Conversion

POST an SCSS file to http://sassas.uk/api/convert and you will be returned the compiled CSS in the response body.

POST /api/convert?apikey=<Your API Key> HTTP/1.1
Host: sassas.uk
Content-Type: application/x-www-form-urlencoded

------Boundary
Content-Disposition: form-data; name="file"; filename="file.scss"
Content-Type: text/css
------Boundary

Grunt Plugin

If you want to use Direct Conversion as part of your Grunt workflow you can use the grunt-sassas plugin.

npm install grunt-sassas

Inline Conversion

Embed your SCSS & SASS files directly into your website without needing to compile them!

Please only use this on development sites as it can be slow.

<link href="http://sassas.uk/api/inline?url=<Encoded URL>" rel="stylesheet">
Example 
<link href="http://sassas.uk/api/inline?url=http%3A%2F%2Fsassas.uk%2Fres%2Fscss%2Fmain.scss" rel="stylesheet">

REST API

The REST API allows you to upload any CSS, SASS or SCSS file, where it will then be converted into the other formats and made available for you to download. These files will be stored for you to retreive or update at a later time

GET /api/resources?apikey=<Your API Key>
POST /api/resources?apikey=<Your API Key>
GET /api/resources/:id?apikey=<Your API Key>
POST /api/resources/:id?apikey=<Your API Key>
PUT /api/resources/:id?apikey=<Your API Key>
